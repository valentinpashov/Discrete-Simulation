# Discrete-Simulation

## Getting Started

Follow these steps to get both the backend and frontend up and running.

### Backend
1.  **Install dependencies**
    ```
    pip install fastapi uvicorn simpy numpy
    ```
2.  **Launch the server**:
    ```
    uvicorn main:app --reload
    ```

### Frontend
1.  **Install the packages**:
    ```
    npm install
    ```
2.  **Ensure visualization libraries are installed**:
    ```
    npm install chart.js react-chartjs-2 axios
    ```
3.  **Start the development server**:
    ```
    npm run dev
    ```
